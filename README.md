# CodeAlpha Hangman Game

## Description

This is a simple Hangman Game developed using Python as part of the CodeAlpha Python Programming Internship.

## Features

- Random word selection
- Letter-by-letter guessing
- Limited number of attempts
- Displays the correct word when the game ends

## Technologies Used

- Python

## How to Run

1. Open the project in Python.
2. Run `hangman.py`.
3. Enter one letter at a time to guess the hidden word.
4. Continue until you guess the word or run out of attempts.
